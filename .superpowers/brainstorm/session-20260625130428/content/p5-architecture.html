三个子模块设计方向

Prophet Forecaster

新建 forecaster.py，封装 Facebook Prophet

  • 月度 GMV / 订单量预测
  • 自动检测节假日效应（双11/618）
  • 输出：趋势、季节性、置信区间
  • 可视化：Plotly 交互图

Chart Reader（增强）

已有 Zhipu GLM-4V → DeepSeek 链路

  • 支持更多图表类型（雷达图/热力图）
  • 批量分析多张图表
  • 输出格式化 Markdown 报告

异常检测增强

从纯统计 → 统计+时序混合

  • 保留 3σ + IQR 规则
  • 新增 Prophet 残差检测
  • 新增 STL 分解异常
  • 多模型投票/加权融合
