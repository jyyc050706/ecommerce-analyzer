Chart Reader 增强设计

当前：单图识别 + 自然语言分析。明确增强边界。

当前现状（ai_layer/tools/chart_reader.py）

视觉模型智谱 GLM-4V
分析模型DeepSeek
输入单张图片路径
输出raw 文本 + NL 分析
图表类型6种（柱/折/饼/漏斗/散点/热力）
Prompt 提取字段类型、标题、轴标签、数据系列、异常点、趋势

增强方向

结构化输出标准

read_chart() 返回真正的 dict（非 raw 字符串）。JSON Schema 定义 + parse 校验 + 降级回退。analyze_chart() 接收结构化 dict 而非 raw 字符串。

批量分析

read_charts(paths: list) → 并发调用 GLM-4V，返回 list[dict]。用于对比多时间段的同类图表。

Markdown 报告输出

analyze_chart() 新增 output_format='text'|'markdown'，返回结构化报告：标题、关键数据、异常标注、结论建议。

Agent 对接

暴露为函数工具，供 agent.py 的自然语言路由调用。用户说"分析这张图"→ agent 自动调 chart_reader。
